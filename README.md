# Bookmark-Manager
A simple web application for adding, viewing, editing, and deleting bookmarks using HTML, CSS, PHP, and MySQL.

![Image Alt](https://github.com/PaTarY0t/Bookmark-Manager/blob/222e76e09fcbe52ede04287a87993e5490985dc9/1.png)

![Image Alt](https://github.com/PaTarY0t/Bookmark-Manager/blob/222e76e09fcbe52ede04287a87993e5490985dc9/2.png)

# Features

Add a new bookmark with a title and URL

Display all saved bookmarks in a clean list

Edit existing bookmarks

Delete bookmarks with a confirmation prompt

Simple and clean CSS design

MySQL database to store bookmarks


# Requirements

XAMPP or any local server with PHP and MySQL support

Modern web browser
